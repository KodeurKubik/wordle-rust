## Installation

You can either download the executable from the RELEASES tab, or compile it yourself by:

1. cloning the repo

```
git clone https://github.com/KodeurKubik/wordle-rust
cd wordle-rust
```

2. running with [cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)

```
cargo run --release
```
